## 1. ダシパック立ち上げ法（世界初）
- 立ち上げ時のアンモニア源としてダシパックをサンプに沈める
- 鰹節（イノシン酸 N×4）、椎茸（グアニル酸 N×5）、昆布（グルタミン酸）→ アミノ酸が窒素源
- パックごと回収可能、溶出が緩やか、腐敗物が散らからない
- 半月で交換、立ち上げ1ヶ月で2パック
- 赤玉土へのミネラルチャージも同時に達成
- 検索結果：世界中で誰もやっていない（2026年3月時点）
- 海外版：MSG + dried shiitake in a tea bag / ブイヨンキューブは塩分過多でNG

## 2. みりん＋無塩料理酒 立ち上げ法
- みりん = エタノール + 糖 + アミノ酸（発酵由来）
- 料理酒 = エタノール + アミノ酸 + 有機酸
- 炭素（C）と窒素（N）を1本で同時供給
- 立ち上げ期 → みりん＋料理酒（C+N）、運用期 → ウォッカ＋酢＋砂糖（Cのみ）
- 注意：必ず無塩タイプを使うこと（塩分はエビに致命的）

## 3. 立ち上げの原則
> **立ち上げ = C + N（炭素＋窒素の同時供給）**
> **運用 = C のみ（窒素は魚が供給）**

## 4. 水槽 = 一つの生命体（臓器アナロジー）
| パーツ | 臓器 | 機能 |
|---|---|---|
| 硝化菌 | 肝臓 | 毒（NH₃）を無毒化（NO₃⁻） |
| ベントスループ | 腎臓 | 老廃物（NO₃⁻→菌体）を排出 |
| オーバーフロー | 心臓 | 血液循環（水流+曝気） |
| 水草 | 肺 | ガス交換（CO₂↔O₂） |
| 赤玉土 | 骨格 | 構造維持+ミネラル供給 |

- 従来の水槽 = 腎臓のない生き物 → 人間が人工透析（換水）
- ホロビオント = 自前の腎臓を持つ生き物 → 自律

## 5. 化学式による全工程の記述
```
立ち上げ: R-NH₂ → NH₃ → NO₂⁻ → NO₃⁻（窒素源から硝化サイクル）
炭素同化: C₂H₅OH + NO₃⁻ + O₂ → 菌体タンパク + CO₂ + H₂O
ベントス: 菌体タンパク → ベントスの体 → 魚の餌
水草: NO₃⁻ + PO₄³⁻ + Fe²⁺ + 光 → 植物体 → トリミングで系外排出
```

## 6. その他の知見
- リン酸塩・重金属の蓄積問題 → 水草のトリミングで系外排出（穴なし）
- ベントスの遺伝的多様性 → イトミミズは無性生殖、魚入替時にベントスも追加で十分
- ポンプ停止体験 → 一晩でサンプ地獄絵図、しかし換水せず復帰（死体もタンパク質）
  - 判断の根拠：「腐敗するから悪い。分解されれば問題ない。好気性分解能力は備わっている。酸素供給だけで行ける」
  - **「好気は全部操作できる」**：嫌気は崩壊が一瞬で原因不明、好気は全サインが目に見えて介入できる
  - ベントスが表層でアップアップ＝酸素不足のサイン→ポンプ再起動で解決
  - 好気性設計の耐障害性：失敗が段階的に見えて、時間的余裕をもって対処できる
- ワイルドアルタム3匹が攻撃性ゼロ → ベントス常時供給で空腹ストレスなし
- 全魚の発色が最高（アルタムの目が真っ赤、ベッコフォルディがブラッドペンシル級）
- 色揚げ飼料も不要（天然ベントスにカロテノイド含有）
- **1年間病気なし・死亡なし**（2026-03-08 確認）
  - アルタムを含む混泳水槽で1年間達成→通常あり得ない実績
  - v7末尾の「腸内フローラ→免疫力向上仮説」の状況証拠として記録価値あり
  - 多様なバクテリアを常時経口摂取＝毎日発酵食品を食べているような腸内環境の可能性
  - 断定はできないが「無換水でも疾病なし」という事実は明確に記録すべき
- 経歴を「約10年」→「約3年」に修正済み
- フレーズ採用：「自然の池は換水しない。じゃあ俺もしない。以上。」
- **サンプで稚魚が自然育成**（2026-03-08 確認）
  - サンプに落ちたミクロラスボラの稚魚を放置→そのまま大人になった
  - サンプ内にマイクロベントス・バクテリア群が稚魚の餌として十分に存在していた証拠
  - 天敵なし・水質安定・給餌不要＝**サンプが天然の育成槽として機能**
  - 繁殖を狙うなら「サンプに稚魚を落として放置するだけ」の**放置繁殖**が成立する可能性
- **炭素源ブレンドの効果**（2026-03-08 感覚ベース・写真なし）
  - 砂糖単体→ブレンド（ウォッカ+酢+砂糖）に変えたところ：
  - 浮遊物は増えたが、**水自体はパキッと透明になった**
  - **管内のヘドロ詰まりが減った**
  - 仮説：複数の炭素源→バクテリアの種が多様化→菌体が均一に小さく分散→フロック化しにくい→管付着少・水透明

## 7. 理論の新規性の本質
> 新しい生物学は何もない。自然が動く「箱」を設計しただけ。
- 新規性は物理的環境の構築（オーバーフロー+サンプ赤玉土+炭素源ブースト）
- 理屈は自然そのもの → だから穴がない
- 生態系の縮小模型（バッファの差を炭素源で補っているだけ）

## 9. 他の無換水システムとの差別化マップ（2026-03-08 調査追記）

### 差別化の核心概念：「アウトプット経路の有無」
> 無換水を「硝酸塩を吸着・蓄積して換水頻度を下げる（減速型）」ではなく、  
> 「食物連鎖でリアルタイムに系外へ出す（代謝型）」で実現している点がホロビオントの根本的差異。

### 比較表：世界の主要な無換水・減换水アプローチ

| システム名 | 地域 | 原理 | 硝酸塩の処理 | 最終的な水換え | 決定的な限界 | ホロビオントとの差 |
|---|---|---|---|---|---|---|
| **ASPシステム**（アクアシステム社） | 🇯🇵 日本 | 専用ソイル吸着＋底面フィルター | ソイルに吸着・保持 | 半年〜1年に1回必要 | ソイルが1年でリセット必要・専用品依存 | OUT経路なし→蓄積限界あり |
| **HONUMIスーパーナチュラルシステム** | 🇯🇵 日本 | 特許技術・微生物生態系（詳細非公開） | 非公開 | 「不要」と謳う | ブラックボックス・検証困難 | 原理不明 |
| **ブルカミアソイル＋底面フィルター** | 🇯🇵 日本 | 高性能多孔質ソイルへのバクテリア定着 | ソイル内での分解 | 大幅削減 | ソイル劣化・定期リセット | ASPと同系統 |
| **高密植水草型（ADA式等）** | 🌐 国際 | 水草が硝酸塩・リン酸を吸収 | 植物体として固定→トリミングで排出 | 理論上減らせる | CO₂添加必須・リン酸等他物質が蓄積（崩壊事例あり） | 一部OUT経路あり、しかし限定的 |
| **Walstad法**（Diana Walstad） | 🇺🇸 英語圏 | 自然底床（赤玉土系）＋高密植・低魚密度 | 水草吸収＋底床バクテリア | 年数回〜無換水 | 低ストッキング必須・ミネラル蓄積リスク | 低負荷設計、OUT経路は水草のみ |
| **アクアポニックス** | 🌐 国際 | 魚の排水で野菜を水耕栽培、浄化された水を戻す | 植物（野菜）が全吸収 | ほぼ不要 | 食用植物が必要・アクアリウムとして成立しにくい | 植物OUT経路は強力、しかし「水槽」ではない |
| **DSB（嫌気砂床）** | 🌐 国際（海水） | 嫌気性細菌が硝酸塩→窒素ガスに還元 | 大気へガス放出 | 理論上不要 | 崩壊時に硫化水素・ブラックボックス | 嫌気=視認不可、リスク大 |
| **ウォッカ法＋プロテインスキマー** | 🌐 国際（海水） | 炭素同化でバクテリア増殖→スキマーで泡として除去 | スキマーが系外排出 | ほぼ不要（海水のみ） | 淡水ではスキマー機能せず（表面張力の差） | **本システムの海水版＝確立済み。淡水版＝ホロビオント** |
| **ホロビオントアクアリウム（本システム）** | 🇯🇵 日本（独自） | 炭素同化＋ベントスループ＋オーバーフロー | ベントス捕食で系外排出 | **ゼロ（足し水のみ）** | OFシステム必要・初心者不向き | **唯一の淡水×好気性炭素同化×生物的OUT経路** |

### 調査から見えた本システムの位置づけ

1. **Walstad法との関係**：欧米で最も普及している自然系無換水だが、低ストッキング・低負荷が前提で、ミネラル蓄積の限界は認識されている。ホロビオントはより高魚密度・高負荷でも成立するシステムとして上位互換的な位置づけが可能。

2. **ウォッカ法との関係**：炭素同化という核心原理はまったく同一。差は「バクテリアの除去手段（スキマー vs ベントス）」のみ。ホロビオントは「淡水でスキマーが使えない問題」をベントスで解決した=淡水版ウォッカ法として明確に位置づけられる。

3. **既存システムが「無換水は難しい」と言う理由**：OUT経路の設計が不完全なため。HONUMIを除くほぼすべてが「蓄積を換水・リセットで解決」というアーキテクチャから出られていない。ホロビオントは「蓄積しない代謝設計」という別のアーキテクチャ。

4. **「足し算・引き算」フレームでの整理**（nature-at-home の問題設定に準拠）：
   - 既存の無換水水草水槽は「引き算が水草トリミング＋吸着材だけ」→蓄積の限界が来る
   - ホロビオントは「ベントス捕食」という第三の引き算を持つ→原理的に蓄積限界がない

### 新規性の文言候補（v13以降用）
> 「淡水における好気性炭素同化＋生物的窒素系外排出の統合システム」
> 「Walstad法の上位互換：ベントスループによる窒素のリアルタイム系外排出を組み込んだ高負荷対応型」
> 「淡水版ウォッカ法：スキマーの代わりにベントスを使うことで初めて成立した、淡水完全自律型生態系水槽」

## 10. 「なぜ誰も思いつかなかったか」考察（2026-03-08 雑談追記）

> 「バクテリアはタンパク質なんだから食えるだろう。なぜその発想が出てこないのか？」

- **思考の枠**：アクアリウム界全体が「バクテリア＝水を浄化する道具」という図式。道具は使われるもので、「食べられるもの」という発想が枠外にある
- **海水が先行した歴史的経緯**：ウォッカ法のフロンティアは海水。スキマーで解決できてしまったため「別解＝生物に食わせる」を考える動機がなかった。**問題が解決されていたから、別解が生まれなかった**
- **ベントスの役割認識の固定**：イトミミズ・ヨコエビは「外から買ってくる魚の餌」。「水槽内で増殖させてバクテリアを食わせる閉ループ」として設計した人がいなかった
- **このフレーズは理論の導入として使える**：「タンパク質なんだから食えるだろう」という当たり前の視点から、なぜ誰も実現できなかったかを逆から語ることで新規性が際立つ

### 文言候補
> 「バクテリアはタンパク質だ。タンパク質は食べられる。では、バクテリアを食べる生き物を水槽に入れればいい。——それだけのことを、なぜ誰もやっていなかったのか。」

## 11. 「新たな系統」として立てる（2026-03-08 雑談追記）

> ウォッカ法との関連性は低い。既存のアクアリウムの枠組みとは機能がかけ離れており、新たな系統として位置づけるべき。

### ウォッカ法との関係の整理
- ウォッカ法の**神髄はスキマーによる除去**（炭素同化→バクテリア増殖→スキマーで泡として系外排出）
- 「炭素を入れる」という一点は共通するが、目的・機構・設計思想が全く異なる
- ホロビオントアクアリウムはむしろ**スキマー除去に依存しないように設計した**のが本質
- 「炭素同化＋ベントス捕食」という別の二段構えであり、ウォッカ法の派生ではない

### 既存のろ過カテゴリと対比
| カテゴリ | 代表手法 | アプローチ |
|---|---|---|
| 物理ろ過系 | 外部フィルター・上部フィルター | 物理的に除去 |
| 生物ろ過系 | 硝化バクテリア中心 | 毒性変換（NH₃→NO₃⁻） |
| 化学ろ過系 | 活性炭・吸着材・ASP | 吸着して保持 |
| 植物吸収系 | Walstad法・ADA式 | 植物に固定→トリミングで排出 |
| 嫌気系 | DSB・脱窒炉 | 窒素ガスとして放出 |
| **生態系設計系（新系統）** | **ホロビオントアクアリウム** | **食物連鎖そのものを設計して代謝させる** |

### 「生態系設計系」の本質
- 「ろ過」の話ですらなく、**生態系の設計**の話
- 「処理する・吸着する・変換する」ではなく、「**代謝させる**」
- これは「アクアリウムろ過の新方式」ではなく、**「アクアリウムとは何か」という定義の更新**に近い
- 水槽を「魚を管理する容器」ではなく「生態系の縮小模型」として再定義する

### 教義の判定基準（2026-03-08 雑談で整理）
> **問題が起きたとき「濾過」で解決せず、「生態系の機能」で解決する**

| 判定 | 基準 | 例 |
|---|---|---|
| ⭕ 許される | 環境に組み込んで放置できるもの | 炭・焼赤玉土・流木・石・淡水海綿 |
| ❌ 教えに反する | 定期的な人間の介入を要求するもの | スポンジ洗い・フィルター交換・活性炭交換 |

> **判定基準：「置いたら放置できるか」**

### 文言候補
> 「ホロビオントアクアリウムは、新しいろ過方式ではない。それは水槽の定義そのものを書き換える試みだ。」

### 方式名の候補（2026-03-08 確認）

| 候補名 | メリット | デメリット |
|---|---|---|
| **ホロビオント式** | 独立系統として明確・理論の独立性を保てる | 入口としての親しみやすさがやや薄い |
| **ウォッカ法ホロビオント式** | 炭素源を使うという操作的共通点を示せる・既知概念が入口になる | ウォッカ法の派生に見えるリスクあり |

> **暫定方針**：「ホロビオント式」を正式名称とし、解説の中で「炭素源としてウォッカ・みりん等を使う点はウォッカ法と手続き上共通する」と注記する形が、独立性と説明容易性のバランスが取れているか。

## 12. バイオスフィアとの接続——概念的位置づけ（2026-03-08 雑談追記）

> アクアリウムというより、バイオスフィア（閉鎖型生態系）の考えが近い。

- **バイオスフィア2**（1991年、アリゾナ）：閉鎖系の中に生態系をまるごと設計し、窒素・水・酸素の循環を自己完結させようとした試み。本システムの発想と構造的に同じ
- **共通する設計思想**：「閉鎖された系の中で、物質循環を設計によって自律させる」
- **スケールの違いだけ**：バイオスフィアは人間を含む閉鎖生態系、ホロビオントは水槽スケールの閉鎖生態系
- **炭素の制御という共鳴**：バイオスフィア2の失敗原因のひとつはコンクリートによるCO₂吸収で光合成が維持できなくなったこと→炭素の流れが系を制御する核心であることは共通
- **正確な言葉**：「アクアリウム」より**「閉鎖型生態系（Closed Ecological System）の水槽スケール実装」**

### 概念的位置づけの更新
```
従来のアクアリウム：魚を飼う → ろ過で水質管理 → 換水で排出
ホロビオント式：  閉鎖生態系を設計 → 物質循環を自律させる → 排出不要
                  ↑ バイオスフィア・宇宙閉鎖系研究と同じ思想領域
```

### 文言候補
> 「これはアクアリウムではなく、卓上バイオスフィアだ。」

## 13. システムの本懐——飼育者の主観的体験（2026-03-08 雑談追記）

> 「飼っている感じじゃない。庭の木を見る感覚」

- 庭の木は誰も「管理してやっている」とは思わない。撮りゟはたまに切るが、それも「付き合い」に近い
- **存在すること自体が風景になっている**——水槽を眺めるのが「飼育確認」ではなく「風景を見る」感覚
- この感覚はシステムが本当に自律していることの証明

### 本懐の言語化
> **「管理することを諦めた結果、管理が不要になった」**

- 換水も、給餐も、CO₂添加も——全て「人間が系を制御しようとする行為」
- ホロビオント式は逆で、「制御を手放して、系に自律させる」設計
- 「飼育」というより**「生態系と同居している」**感覚

### 文言候補
> 「水槽は今日も静かに、自律して動いている」（v7最終記）——この一文の内実が、庭の木を見る感覚と完全に一致している。

## 14. 教育的価値——「全部が化学式に見えてくる」（2026-03-08 雑談追記）

> 「この水槽の構築って自然界に発生することそのものだから、勉強にすごくいい。全部が化学式に見えてくる」

- このシステムを構築・観察することは、**生態系の物質循環を実物で学ぶ**こと
- NH₃→NO₂⁻→NO₃⁻の硝化、炭素同化、食物連鎖——教科書の概念が目の前で起きている
- 白濁＝バクテリアブルーム、クリア化＝生態系の安定化、ヘドロ＝菌体タンパクの蓄積……**全現象が化学・生物学的プロセスとして読める**
- 変態アクアリストにしか構築できない一方で、**理系的知的好奇心への訴求力は高い**
- 「勉強になる」という側面は対象読者の拡張につながる可能性：理系学生・生物教師・研究者など

### 文言候補
> 「水槽を眺めるたびに、自然界の化学式が動いているのが見える。」

### 逆説的教育論：「初心者がまずやるべきシステム」

### 派生案：極酸性ベントスループ×チョコグラ水槽（2026-03-08 雑談）

> 「江酸性ベントスループが成り立つなら、チョコグラが簡単に飼えるようになる」

**理論的根拠**
- 従属栄養バクテリアはNH₄⁺を直接然体へ取り込める（「同化的窒素除去」）→硭化を経ずに窒素処理可能
- pH5台でも従属栄養バクテリアは機能する可能性がある
- ベントス候補：イトミミズ（酸性耐性あり）・ミズムシ（強い）『ミジンコは厳しい』

**リン・重金属の排出問題→ポトスアクアポニックスで解決**
- ポトスはpH5台でも水耕栽培可能・成長速い・重金属吸着実績あり
- サンプ上部に垂らして根を淫けるだけのシンプル構造
- 葉が伸びたら切る→リン・重金属を系外に排出

```
窒素： 酸性ベントスループで系外排出（換水不要）
リン・重金属： ポトスアクアポニックスで吸収→トリミングで系外排出
結果： 完全無換水チョコグラ水槽（理論上）
```

- チョコグラの原産地（マレーシア・スマトラの泥炭湟地）もブラックウォーター→性質と近い
- **「等酸性ベントスループ×チョコグラ」は世界初の機能水槽となる可胻性がある**

## 20. pH理論——底床で決まる（2026-03-08 雑談）

> 「底床の性質がわかっていればpH調整すごく簡単」

- **フルボ酸（マジックリーフ）が換水で薄まらない**→ブラックウォーター環境の安定維持にも有利

## 窒素管理の体系化（2026-03-09 深夜 考察）

### こすりつぶし対決の結果（後半）

| 反論 | 結果 |
|---|---|
| ③ 系外排出の証拠がない | 表現修正→「高次循環による無毒化」が正確 |
| ④ 生存者バイアス | 「じゃあ追試してケロ」。GitHub Pagesに教典あり |
| ⑤ 全部化学式で追えるは嘘 | 「追える範囲で全部追った」が正確。ミクロは追えないが診断には十分 |

- **③の表現修正から理論の本質が逆にクリアになった**
- 「系外排出」→「窒素を毒性ある形態から無毒な形態へ変換し続ける」が正確
- ベントスが「系外に排出」しているのではなく「高次バイオマスに変換・循環」している

---

### アクアリウム窒素管理の3分類（新体系）

```
Ⅰ. 排出型（Nitrogen Export）
   「窒素を系外に捨てる」
   ├── 物理排出：換水
   ├── ガス排出：DSB・外部脱窒炉
   └── 生物排出：Walstad・ADA（水草トリミング）

Ⅱ. ハイブリッド型（Hybrid）
   「循環＋排出の混合」
   └── ビオトープ（藻類循環＋抽水植物排出）
       ※屋外ビオトープは抽水植物のNボリュームが大きく、排出が支配的

Ⅲ. 循環型（Nitrogen Cycling）　←【新カテゴリ】
   「窒素を系内で回し続ける。毒性ある形態→無毒なバイオマスに変換」
   ├── 従属栄養型：ホロビオント式（炭素源＋ベントスループ）
   └── 光独立栄養型：グリーンウォーター式（藻類＋動物プランクトン）
```

- **「濾過から代謝へ」**がパラダイムシフトの本質
- ホロビオント式は「濾過をしないアクアリウム」——フィルターはあるが、やっていることは代謝
- ビオトープの思想（屋外）を屋内・室内・高負荷環境に移植したのがホロビオント式

---

### 循環型の2軸マトリクス

|  | 低負荷（メダカ等） | 高負荷（大型魚・密飼い） |
|---|---|---|
| **独立栄養（光）** | グリーンウォーター式 ✅ | ❌（光量が追いつかない） |
| **従属栄養（炭素源）** | ホロビオント式（過剰スペック） | **ホロビオント式 ✅（本領発揮）** |

- **高負荷×循環型 = 従属栄養しかない**——生化学的制約
- 炭素源を人工投入することで24時間・室内・高負荷に対応できる
- グリーンウォーター式は光量・温度依存で制御が難しい

---

### 循環型の3軸（エネルギー源別）

| 軸 | エネルギー源 | 代表例 | 実用性 |
|---|---|---|---|
| ① 従属栄養型 | 有機炭素（炭素源） | ホロビオント式 | ✅ 現実的 |
| ② 光独立栄養型 | 光（光合成） | グリーンウォーター式・ビオトープ | ✅ 屋外限定 |
| ③ 化学独立栄養型 | 無機化合物（H₂S等） | 嫌気チャンバー硫黄駆動型 | ❌ 理論上のみ |

- 生化学的に**エネルギー源は3種類のみ**。これ以上の軸は存在しない
- ③は自然界では深海熱水噴出孔（ハイドロサーマルベント）に実在する
- 家庭での③実装は人体へのH₂S毒性リスクがあり非現実的（核融合炉枠）

**発電所アナロジー：**
- 藻類式 = 太陽光発電（クリーン・天候依存・低出力）
- ホロビオント式 = 石炭発電（安定・燃料投入必要・現実的）
- 嫌気チャンバー硫黄式 = 核融合炉（理論上最高効率・制御困難・人体リスク）

### 元素別役割分担（重要な整理）

| 元素 | 処理方法 | 担当コンポーネント |
|---|---|---|
| **窒素（N）** | ベントスループで無毒化循環 | 従属栄養バクテリア＋ベントス |
| **リン（P）** | 吸着・水草トリミングで排出 | 焼赤玉土（鉄分がPO₄³⁻を吸着）＋水草 |
| **重金属** | 上水道品質に依存（日本では実害レベルに達しない可能性） | 焼赤玉土吸着＋水草（保険） |

- 水草は「N排出のため」ではなく「P・重金属排出のため」に置く
- 日本の水道水品質が高いため、重金属蓄積は実害レベルに達しない可能性がある
- **水草なしホロビオント式**も窒素管理上は成立する。P管理のみ追加検討が必要
- 「水草は保険であって必須ではない可能性がある（日本の上水道前提）」
- 国・地域の水道水品質によって重要度が変わる点を注記すること

### 副反応としての嫌気効果（DNRA）

- ホロビオント式のベントス底床内で**微小嫌気層**が自然形成される可能性
- DNRA（dissimilatory nitrate reduction to ammonium）：NO₃⁻→NH₄⁺（系内に留まる）
- DNRAはC:N比が高い環境で優位——ホロビオント式はまさに高C:N環境
- 意図せず**好気性炭素同化（主）＋DNRA（副）の二重循環**が起きている可能性
- ベントスが底床を掘り返すことで好気・微小嫌気が共存する層が形成される
- **設計していないのに自然に最適化されていく**性質の一例

### ベアタンク×ホロビオント式（派生型）

- ベアタンク（底床なし）派のディスカス・アロワナ飼育者向け
- **メインタンク：ベア（底床なし）のまま**
- **サンプのみ：焼赤玉土＋ベントス**
- OFで糞・有機物がサンプに落下→ベントスが処理→ベントスが魚のおやつとして循環
- 「糞がおやつに変わって戻ってくる」自動化
- 給餌量を細かく管理したい人には向かない（ベントスが勝手におやつ供給）
- まめな人は従来どおり掃除できるので問題なし→自己選別が働く

### 嫌気チャンバー硫黄駆動型（ロマン枠・記録のみ）

```
理論的シーケンス：
嫌気チャンバー（密閉）：
  硫酸塩還元菌：SO₄²⁻ + 有機物 → H₂S
  
化学独立栄養チャンバー（隔離）：
  硫黄酸化菌（Thiobacillus等）：H₂S + O₂ → SO₄²⁻ + エネルギー
  このエネルギーでNO₃⁻ → 菌体バイオマス（窒素循環）

副産物：微生物燃料電池（MFC）で微弱発電（現状：LEDをかろうじて光らせる程度）

課題：ポンプ稼働には現状の発電量の100万倍が必要
```

- **炭素源・光・外部電力すら不要の完全自律型**（理論上）
- 自然界の実例：深海熱水噴出孔（ハイドロサーマルベント）生態系
- 家庭実装：H₂S毒性リスク（人体への影響）があり非現実的
- MFCがポンプ稼働できるレベルに達すれば宇宙水槽・宇宙基地応用が可能
- **当面はロマン枠として理論に記録しておく**
