🧬 ホロビオントアクアリウム理論

炭素同化と食物連鎖による淡水水槽の窒素循環理論

ホロビオントアクアリウム理論——従属栄養細菌の炭素同化と食物連鎖による淡水水槽の窒素循環

淡水オーバーフロー水槽における無換水・無給餌・CO₂無添加の1年間維持記録と理論的考察

📄 概要

本稿では、従属栄養細菌によるアンモニアの直接同化と、生成されたバイオマスをデトリタス食性生物に捕食させて肉食魚に供給する食物連鎖を組み合わせることで、淡水水槽における窒素循環を硝化サイクルに依存せずに実現する理論（ホロビオントアクアリウム理論）を提案する。本理論に基づき、オーバーフロー式水草水槽を換水ゼロ・給餌ゼロ・CO₂添加ゼロで1年以上にわたり安定維持した実証結果を報告する。

⚠ 前提条件
  • オーバーフロー式水槽が必須（好気性細菌の酸素要求量を曝気で満たすため）
  • 立ち上げ時に約1ヶ月間のバクテリアブルーム（白濁）が発生する
  • 中〜上級者のアクアリスト向け（原理の理解と経過観察の能力が必要）

導入

淡水アクアリウムにおいて、定期的な換水は硝酸塩の蓄積とpH低下を抑制するための標準的な手法である。しかし、大型水槽では換水作業が大きな維持管理コストとなり、趣味としての継続を困難にする要因となりうる。

本稿で提案するホロビオントアクアリウム理論は、炭素源の投入により従属栄養細菌にアンモニアを直接同化させ、生成されたバイオマスをデトリタス食性生物が捕食し、それをさらに肉食魚が捕食する——という食物連鎖を通じて、水槽内の窒素を循環させるアプローチである。

Holobiont（ホロビオント）とは、宿主と共生微生物群全体を一つの超個体として捉える生物学の概念であり、本システムの設計思想を表す名称として採用した。

淡水の構造的制約

海水アクアリウムでは、ウォッカ法（炭素源の投入）で増殖した菌体バイオマスをプロテインスキマーで系外排出できる。しかし、淡水では表面張力の違いによりスキマーが機能しない。この構造的制約が、淡水での炭素同化アプローチを困難にしてきた。

比較項目 海水（ベルリン式＋ウォッカ法） 淡水（本理論）

2. 従属栄養細菌によるアンモニア直接同化

C:N比と同化条件

従属栄養細菌が窒素をどのように処理するかは、環境中のC:N比（炭素と窒素の比率）に大きく依存する。

微生物学的背景
  • C:N比が低い環境（通常の水槽）→ アンモニアは余剰となり、硝化菌による酸化的処理（NH₄⁺ → NO₂⁻ → NO₃⁻）が進行
  • C:N比が高い環境（炭素源投入時、目安 C:N > 約20）→ 従属栄養細菌が炭素源をエネルギーとしてアンモニアを直接、自身の細胞タンパクに同化

すなわち、炭素源を投入してC:N比を十分に高めることで、アンモニアの処理経路を硝化（酸化）から同化（菌体取り込み）へ切り替えることが可能となる。

増殖速度の差——なぜ硝化サイクルがバイパスされるか

この経路切り替えが成立するもう一つの条件は、従属栄養細菌と硝化菌の増殖速度差にある。

細菌群 世代時間 NH₄⁺処理方式 副産物
従属栄養細菌 約20分〜数時間 同化（菌体タンパクに取り込む） CO₂、菌体バイオマス
硝化菌（Nitrosomonas等） 約12〜24時間 酸化（NH₄⁺ → NO₂⁻ → NO₃⁻） H⁺（→ pH低下）、NO₃⁻

十分な炭素源が存在する条件下では、従属栄養細菌が硝化菌より桁違いに速く増殖し、利用可能なアンモニアを先に消費する。結果として、硝化サイクル（NH₄⁺ → NO₂⁻ → NO₃⁻）は事実上バイパスされる。

3. デトリタス食性生物による窒素循環

理論原理

前節で述べた通り、従属栄養細菌はアンモニアを菌体バイオマスに変換する。しかし、これだけではバイオマスが水槽内に蓄積し続ける。本理論の第二の柱は、このバイオマスをデトリタス食性生物に捕食させ、さらにそのデトリタス食性生物をメインタンクの肉食魚に捕食させることで、窒素を食物連鎖内で循環させることにある。

この理論において、デトリタス食性生物の種類は原理上問わない。イトミミズ、ヨコエビ等の微小なベントスでも、ドジョウのような大型の底生魚でも、理論的には同じ役割を果たす。

循環の手段——フルオートと手動

デトリタス食性生物をメインタンクに戻す方法は、実装上の選択肢となる。

方式 デトリタス食性生物 メインタンクへの移送 捕食者
フルオート イトミミズ・ヨコエビ等（微小） ポンプを通過して自動供給 小〜中型魚
手動 / 大型ポンプ ドジョウ等（大型） 人の手で移送 or 大型ポンプ 大型肉食魚

本稿で報告するシステムでは、家庭用ポンプの現実的なサイズを考慮し、微小なベントス（イトミミズ・ヨコエビ等）によるフルオート方式を採用した。

4. 理論から導出される帰結

上記のメカニズム——炭素同化によるアンモニア直接同化＋デトリタス食性生物による食物連鎖——から、以下の帰結が理論的に導出される。

🍽️
帰結①：無給餌
サンプで増殖したデトリタス食性生物がメインタンクの魚の餌となる。フルオート方式では、ポンプを通じて常時自動供給される。
💨
帰結②：CO₂無添加
好気性従属栄養細菌の代謝副産物としてCO₂が常時産生される。この量はオーバーフロー落水によるガス抜けを上回り、水草の光合成に十分なCO₂を供給しうる。
♾️
帰結③：底床非嫌気化
デトリタス食性生物（ベントス等）が底床を物理的に撹拌し続けるため、底床の嫌気化（硫化水素発生）が抑制される。リセットの必要性が低減する。

帰結④：pHロック

本理論の特筆すべき帰結として、pHロック現象がある。そのメカニズムは以下の因果連鎖で説明される。

pHロックの因果連鎖

① 炭素源投入 → C:N比上昇
② 従属栄養細菌がNH₄⁺を直接同化 → 硝化が非進行
③ 硝酸（HNO₃）が生成されない → pH低下の原因が消失
④ pHは底床素材の緩衝能が決める化学平衡点で安定

重要な点として、底床はシステムの前提条件ではなく、pHの平衡点を決めるパラメータである。サンプに多孔質素材（バクテリアの定着面）さえあれば本理論は機能し、メインタンクはベアタンクでも理論上成立する。

原理

従来の水槽では、硝化サイクルが進むと硝酸（HNO₃）が蓄積し、pHは時間とともに低下する。この酸性化が換水を必要とする根本原因の一つ。
本システムでは従属栄養細菌がNH₄⁺を硝化させずに直接同化するため、硝酸が生成されない。酸の供給源が消えることで、pHは底床素材の緩衝能（バッファー容量）が決める化学平衡点で停止する。
焼赤玉土ならpH 6.85、ソイルならpH 5台後半、サンゴ砂ならpH 8台。底床を選ぶことで、目標pHを設計できる。

3種の炭素源を組み合わせることでモノカルチャー化を防ぎ、多様なバクテリアコミュニティを維持する。

⚠️ 注意
バナナは必ず可食部（果肉）のみ使用すること。皮には農薬が残留している場合があり、エキスに溶け出してエビ類が全滅する危険がある。

本エキスをサンプへ定量（目安：ポンプ1プッシュ）を毎日投入する。ドーシングポンプによる完全自動化も可能。ウォッカのアルコールが防腐剤として機能するため、常温保存が可能である。

立ち上げ手順（約1ヶ月）

1
底床の選定
焼赤玉土（硬質）を推奨する。多孔質でバクテリアが定着しやすく、栄養分がないため初期水質が安定しやすい。
📌 重要
必ず「硬質」タイプを選ぶこと。通常の赤玉土は水中で崩れる。
2
サンプへの赤玉土充填
サンプ内にも焼赤玉土を可能な限り敷き詰める。ポンプとは区画を隔てること。バクテリアの定着面積を最大化する。
3
空回し＋初回投入
水を入れてオーバーフローを空回し。ゴーストフィーディング（餌を少量投入してアンモニア源を供給）と炭素源エキス投入を開始する。数日以内に白濁が始まる。
🚫 立ち上げ期間の禁止事項
換水・白濁除去剤の使用
白濁はバクテリアブルームであり、系が正常に進行していることを示す。介入すると従属栄養細菌のコロニー形成が初期化される。
4
1ヶ月継続 → 突然のクリア化
白濁したまま毎日投入を継続する。約1ヶ月後、水が急速に透明化することが観察される。これはバクテリアのフロック形成と底床への定着が完了し、系が安定状態に移行した指標である。理論上、十分な酸素と炭素源が確保された時点で従属栄養細菌によるアンモニア同化が成立するため、この段階から生体投入が可能となる。

適用条件と制約

必須条件
  1. 新規立ち上げ時にのみ実践すること。立ち上げ済みの水槽への後付けは高リスク。
  2. オーバーフロー式が必須。外部フィルターは一瞬でヘドロが詰まるため特に危険。
向かない人
  • ベントスへの嫌悪感がある人：サンプにはイトミミズ・ヨコエビ・ミズムシ等が大量生息し、メインタンクにも流入する。
  • アクアリウムのお世話を楽しみたい人：システムが安定すると、やることがほとんどなくなる。
オプション
  • 浮遊物が気になる場合：サンプに淡水海綿を入れることで生物的に除去できる
  • 投入の完全自動化：ドーシングポンプの活用

10. 考察と展望

大型水槽への適用可能性

大型水槽では水量が増えるほど水質バッファが大きくなり、変動が起きにくい。バクテリアとデトリタス食性生物の生息域が広がることでシステムの安定性が向上すると考えられる。従来の換水前提の管理では「大型化するほど維持が困難」であるが、本理論ではその関係が逆転する可能性がある。

限界と未検証事項

  • 本稿の実証は1系統のみであり、再現性の検証は今後の課題である
  • 底床素材の違いによるpHロックへの影響は理論上の予測であり、ソイルやサンゴ砂での実証は行っていない
  • ドジョウ等の大型デトリタス食性生物での運用例はない
  • 水草レイアウト水槽（ADA式等）への応用は未検証だが、理論上は適用可能と考えられる

まとめ

本稿では、従属栄養細菌によるアンモニアの炭素同化と、デトリタス食性生物を介した食物連鎖による窒素循環を組み合わせることで、淡水水槽における換水不要のシステムを実現する理論（ホロビオントアクアリウム理論）を提案し、その1年間の実証結果を報告した。

本理論の核心は以下の2点である：

  1. 炭素源投入によるC:N比の上昇 → 従属栄養細菌によるアンモニア直接同化 → 硝化バイパス
  2. バイオマスをデトリタス食性生物が捕食 → メインタンクの肉食魚が捕食 → 窒素が食物連鎖内で循環

この理論は、デトリタス食性生物の種類、底床素材、循環方式といったパラメータから独立しており、様々な環境への適用可能性を持つと考えられる。今後の検証とフィードバックを期待する。
