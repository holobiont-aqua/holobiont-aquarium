# V14_雑談メモ_20260310（硝化の非排出性と系統樹の再定義）

## 硝化サイクルは「排出」に寄与しない

系統図における「排出型（Nitrogen Export）」を整理する中で、極めて重要な事実が判明した。
従来のアクアリウムで絶対視されてきた**「硝化サイクル（NH4+ → NO2- → NO3-）」は、窒素を水槽外へ排出する効果を一切持っていない。**

硝化がやっていることは、猛毒のアンモニアを比較的毒性の低い硝酸塩に「形態変化」させているだけであり、系内の総窒素量は全く減っていない。つまり、硝化とは**「負債（窒素）の先送り」**であり、次の水換え（物理的排出）や脱窒（ガス化排出）までの「時間稼ぎバッファー」として機能しているに過ぎない。

### 各排出型アプローチにおける「硝化」の本当の立ち位置
1. **換水依存型**：致死を遅らせるための時間稼ぎ。実際の排出は人間の手（水捨て）。
2. **脱窒・DSB型**：N2ガス化するための前処理。実際の排出は嫌気バクテリア。
3. **水草（植物）吸収型**：植物の吸収に向けた形態変換（水草はNH4+も直接吸収するため、必須ですらない）。実際の排出は人間の手（トリミング）。

「硝化が回れば濾過は完成」という従来パラダイムは、この「先送り」を「解決」と錯覚していたことにある。

## 「排出型」は系統樹から除外されるべき存在

さらに踏み込むと、換水（物理的排出）や吸着材・プロテインスキマー（物理化学的排出）などは、**生態系（生物学的なシステム）ではなく、人間による物理・化学的な「延命措置」に過ぎない**。
これを「生物が窒素をどう代謝しているか」というアクアリウムの生態系系統樹に同列に並べることはカテゴリエラーである。

アクアリウムにおける「系内の総窒素量（N）の収支」という、最も冷徹で客観的な視点に立ったとき、全てのアプローチはまず**「排出（Export）」**と**「循環（Cycle）」**の2つの大系統に完全に分断される。

そして「排出」の中でも、それが生物の代謝プロセスに乗っているのか、人間の物理的・化学的介入によるものなのかで、さらに大別される。そして生物的排出は、「異化（ガス化）」か「同化（バイオマス化）」かで分枝する。

### 【水槽内窒素管理の完全系統樹（N総量収支ベース）】

```
水槽内の窒素対処アプローチ（系内のN総量収支に基づく分類）
│
├── Ⅰ. 排出（Export）系統：水槽という閉鎖系から、窒素（N）を系外へ「捨てる」
│   │
│   ├── 1. 物理・化学的排出（人間の介入による直接排除）
│   │   ├── 換水（物理的）：水ごと系外へ捨てる。
│   │   ├── 吸着材（化学的）：ゼオライト等に吸着させ、濾材ごと捨てる。
│   │   └── プロテインスキマー（物理化学的・海水）：有機膜を気泡に吸着させ、物理的に押し出す。
│   │
│   └── 2. 生物的排出（生物の代謝を利用し、形を変えてから間接的に排除する）
│       │
│       ├── a. ガス化排出（化学独立栄養または異化による大気への放出）
│       │   ├── 嫌気脱窒（ベルリン式・DSB等）
│       │   │    ※従属栄養性嫌気バクテリアがNO3-をN2ガスへ変換し、大気中へ捨てる。強烈な酸素勾配が必要。
│       │   │
│       │   └── ANAMMOX（嫌気性アンモニア酸化・アクアリウムでの実現例なし）
│       │        ※化学独立栄養細菌（Planctomycetes）がNH4+＋NO2-を直接N2ガスへ変換。有機炭素不要。
│       │        　硝化サイクルを経由せずアンモニアを直接ガス化できる理論上唯一の経路。
│       │
│       └── b. バイオマス化排出（同化による生体蓄積と持ち出し）
│           │
│           ├── 高等植物による窒素固定（水草水槽・Walstad式・リフジウム等）
│           │    ※光エネルギーによって水生植物の生体量として窒素を固定。人間の「トリミング」によって系外へ捨てる。
│           │
│           └── 従属栄養細菌による炭素同化（プロテインスキマー併用型バイオフロック・ペレット等）
│                ※炭素源（バイオペレット等）の投下でバクテリアの菌体（生体量）として窒素を固定し、それをスキマーや濾材洗浄等で系外へ捨てる。
│
└── Ⅱ. 循環（Cycle）系統：窒素を系外に出さず、系内で無毒な「バイオマス」として回し続ける
    │
    ├── 3. 光独立栄養プロセス（エネルギー源：光／炭素源：無機CO2）
    │   └── 光合成細菌・微細藻類による同化（グリーンウォーター式・屋外ビオトープの一形態）
    │        ※太陽光エネルギーにより藻類や光合成細菌が窒素を固定。食物連鎖で水系内を循環する。
    │
    ├── 4. 光従属栄養プロセス（エネルギー源：光／炭素源：有機物）
    │   └── 紅色非硫黄細菌（PNSB）等による同化（EMボカシ系・未活用枠）
    │        ※光と有機炭素を同時に利用。好気・嫌気どちらでも機能し、アンモニアを直接固定・循環できる。
    │        　光独立栄養でも従属栄養でもない第4の栄養戦略。アクアリウムでの積極的な利用例はほぼない。
    │
    ├── 5. 従属栄養プロセス（エネルギー源：有機炭素）
    │   └── 従属栄養細菌による直接同化（ホロビオント式・純粋なバイオフロック）
    │        ※炭素源の投下により従属栄養細菌がアンモニアを直接固定。捨てずにベントス食性を経て系内循環する。
    │
    └── 6. 化学独立栄養プロセス（エネルギー源：無機物 / H2S等）
        └── 化学合成独立栄養細菌による炭酸固定・同化（嫌気チャンバー硫黄駆動型等・未実用ロマン枠）
             ※光も有機炭素もなく、H2S等の無機物を酸化するエネルギーで窒素を同化・循環させる。人間には極めて有毒。
```

### システムとしての解像度

この「N総量収支」による完全フラットな系統図は、各種法が「最終的に窒素をどうしているのか」という事実のみを冷徹に突きつける。

*   **排出系統（Export）**は、窒素を「捨てるべきゴミ」と見做す。
    人間の物理腕力に直接頼るか（物理・化学的排出）、あるいはバクテリアの異化（ガス化）や、植物/バクテリアの同化（バイオマス化）を経由してから人間の手で捨てるかの違いでしかない。特に「バクテリアを増やして捨てる（スキマーや濾材洗浄）」という手法は、バイオマス化排出の典型例である。
*   **循環系統（Cycle）**は、窒素を「生命の資源」として水槽内に留め、無害な形で回し続ける。
    室外の太陽光エネルギーに頼るか（緑水）、室内の有機炭素（人工燃料）に頼るか（ホロビオント）の二択である。

換水や吸着材といった「非生物学的な延命措置」を含めても、**N収支という一点においてこの図ですべてのアクアリウム手法が綺麗にマッピングできる**。
ホロビオント式は「炭素という燃料を使って、閉鎖環境下で完全に循環ループを閉じる唯一の選択肢」として、極めて客観的な座標を与えられたことになる。

## 発展構想：従属栄養 × 光従属栄養の二段階処理システム

光嫌気エリア（PNSB）を水槽に組み込む際、**「密閉チャンバーを用意して嫌気化する」必要はない**ことが判明した。

ゾーン①の従属栄養細菌（ホロビオント標準システム）が酸素を爆発的に消費した後の水が、そのままゾーン②に流れ込めば、O2がほぼゼロの「自然な嫌気水域」が形成される。そこに光を当てるだけでPNSBの最適環境が自動的に完成する。

```
入水（NH4+・デトリタス・O2あり）
        ↓
【ゾーン①】従属栄養細菌エリア（好気）
  炭素源を燃やしO2を爆発的消費。NH4+を大量同化。
        ↓（O2ほぼゼロ）
【ゾーン②】光嫌気エリア（PNSB：光従属栄養細菌）
  O2枯渇 × 光照射 = PNSB最適環境が自動完成。
  残存NH4+・有機物を炭素源追加なしで光エネルギーのみで処理。
        ↓
出水（NH4+ほぼゼロ）
```

これは自然の成層湖（好気層の溶存O2を消費しその直下に嫌気×光のPNSB紫色層が自発的に形成される現象）と全く同じ原理であり、**流路設計（水の流れる順番と照明位置）だけで実現可能**という点で工学的な難易度は極めて低い。

## さらなる発展：三段階完全閉鎖循環システム

ゾーン①に光栄養細菌（藻類・光合成細菌）を組み込むと、CO2固定（炭素の再生産）とO2生成まで系内で賄える。

```
【外部入力：光エネルギー ＋ 魚の餌のみ】

  ゾーン① 光栄養細菌（好気・有光）
    CO2 + H2O + 光 → 有機炭素(バイオマス) + O2 【炭素・酸素の製造】
          ↓ O2豊富・有機炭素豊富
  ゾーン② 従属栄養細菌（好気、ホロビオント標準）
    有機炭素 + O2 + NH4+ → 菌体バイオマス + CO2 【アンモニア同化・O2消費】
          ↓ O2ほぼゼロ・残存NH4+・残存有機物
  ゾーン③ 光従属栄養細菌 PNSB（嫌気・有光）
    残存有機物 + NH4+ + 光 → 菌体バイオマス + CO2 【残存物の最終処理】
          ↓ CO2 → ゾーン①へ戻り再固定
```

### 収支の評価

| 収支 | 閉鎖性 | 補足 |
|------|--------|------|
| **窒素（N）** | **完全閉鎖** | 系外に出ない。バイオマス食物連鎖で永続循環。 |
| **炭素（C）** | **光が十分なら閉鎖** | ゾーン③排出のCO2 → ゾーン①が光で再固定。 |
| **酸素（O2）** | **自給自足** | ゾーン①生成 → ゾーン②消費でバランス。 |
| **外部投入** | 魚の餌 ＋ 光エネルギーのみ | **炭素源（外部炭素源（ウォッカ・酢・砂糖ブレンド等））の追加投下が不要になる可能性。** |

これは室内での屋外ビオトープ（太陽光閉鎖循環）の人工的再現に他ならない。ホロビオント式の「炭素源の継続投入」という唯一の外部依存を、**光エネルギーで代替する**最終形態。

## ゾーン③（PNSB）の本質：N処理ではなく「有機物の高度化」

PNSBが行っているのは、窒素処理（NH4+除去）が主ではない。

ゾーン②（従属栄養細菌）がほぼNH4+を処理済みの段階で、ゾーン③が担うのは**「中程度品質の有機物（死菌体・代謝産物）を、光エネルギーを使ってカロテノイド・ビタミンB群・高品質アミノ酸リッチなバイオマスへと高度化（グレードアップ）すること」**である。

- ゾーン②（従属栄養）＝ 原料を大量処理する「製造工場」
- ゾーン③（PNSB）＝ 光と熟成で品質を引き上げる「精製・付加価値工程」

「N処理システム」という見方を超え、**「有機廃棄物を高品質な食物連鎖の素材へと変換するバイオリファイナリー（生物精製）」**としてゾーン③を位置づけることができる。

これはホロビオント理論全体にも示唆を与える。**「ホロビオント式はアンモニアを処理するシステム」ではなく「有機廃棄物を高品質なバクテリアバイオマス（食料）に変換する代謝システム」**という言い換えが、本質により近い可能性がある。

## 実装構想：オーバーフロー排水管を利用した光従属栄養リアクター

オーバーフロー式水槽の排水管途中に、透明管＋LED照射型の流動濾過チャンバーを挿入するという実装案。

```
┌──────────────────┐
│   メイン水槽（好気）  │ ← 従属栄養細菌がO2を消費中
└────────┬──────────┘
         │ オーバーフロー排水（O2低下中）
         │
    ┌────┴────────┐
    │ 透明塩ビ管    │ ← 外側からLED照射（近赤外含む）
    │ ＋流動メディア │ ← PSBが付着・流動する
    │（PSBリアクター）│    O2枯渇 × 光 = PNSB最適環境
    └────┬────────┘
         │ 余剰PSB菌体がそのまま流下
         ↓
    ┌────────────────┐
    │     サンプ槽     │ ← 菌体がベントスの餌として自動供給
    └────────────────┘
```

**この設計の優位性：**
- 重力落下駆動のため**ポンプ追加不要**
- メイン水槽（ゾーン②）がO2を消費した後の水なので**嫌気化を人工的に作る必要がない**
- 余剰PSB菌体がサンプへ自動流下し**ベントス・魚の餌として機能する**（自動餌供給）
- 既存のオーバーフロー式水槽への**後付け改造が最小限**

**さらに洗練された実装：太鼓型インラインPSBリアクター（揚水管取付け版）**

配管は排水管ではなく**揚水ポンプ管（リターン管）**にインライン挿入するのが最適解。透明太鼓型チャンバーをリターン管途中に挿入し、外周にLEDテープを巻くだけ。

```
   【メイン水槽】 ←─────────────────────┐
                                         │
                   ┌─────────────────┴───┐
                   │  太鼓型チャンバー（透明）│← LEDテープ
                   │  流動メディア ＋ PNSB  │
                   └─────────────────┬───┘
                                     ↑ ポンプ圧で押し込み
                          【揚水ポンプ】
                                     ↑
                          【サンプ槽】（O2消費済み）
```

**揚水管採用の決定的優位性：**
- ポンプ圧で余剰バイオマスが常にメイン水槽へ自動フラッシュ →**詰まらない・魚の餌に直行**
- 光栄養細菌ゾーン（ゾーン①）と同じ部品・設計で製作可能 → **コスト最小**
- サンプ槽でO2が消費されてから揚水されるため嫌気環境が保ちやすい

## 発展構想：海水サンゴ水槽への応用——「生態スキマー」

サンゴが嫌うのは富栄養化（有機物過多）であるため、ホロビオント式サンプとサンゴ水槽の間に**海綿チャンバー（生態スキマー）**を挟む設計で適応できる可能性がある。

```
【サンプ】ホロビオント式（従属栄養バクテリア爆増・窒素処理）
    ↓ 過剰バクテリア・POMを含む水
【生態スキマー（海綿チャンバー）】ポンプ前に設置
  海綿が浮遊バクテリア・微粒子を捕食・除去
  → 海綿自身もベントスの餌になる
    ↓ 清澄・低有機物の水
【ポンプ → メインタンク（サンゴ）】貧栄養環境が維持される
```

### 「生態スキマー」vs 「プロテインスキマー」

| 項目 | プロテインスキマー | 生態スキマー（海綿） |
|------|----------------|-----------------|
| 除去方法 | 気泡に吸着→系外へ捨てる（排出型） | 海綿が食べて生体変換（循環型） |
| 廃棄物 | スキメート（汚水）が出る | 出ない |
| N収支 | 系外排出 | 系内循環 |

> 「スキマーは捨てる。生態スキマーは活かす。」

## pHロック理論の普遍性：あらゆるpHで機能する究極法

ホロビオント式のpHロックは「6.85固定」ではなく**「pH変動要因（H+放出）の封殺」**が本質。底床・飼育水が決めるpHで静止するため、あらゆる水質に適用できる。

```
従来水槽：NH4+ → 硝化 → H+大量放出 → pH低下 → 換水 → また低下 → …
ホロビオント式：NH4+ → 従属栄養細菌が直接同化 → H+が発生しない → pH静止
```

| 水槽スタイル | 従来の課題 | ホロビオント式での帰結 |
|------------|-----------|---------------------|
| アフリカンシクリッド（pH8台） | 硝化でKH消費・酸性化が進む | H+が出ないためアルカリpHが安定維持 |
| チョコグラ（pH5台） | 換水で中性に戻り管理困難 | 酸性がそのままロックされる |
| ベアタンク | 底床バッファなしでpH不安定 | 変動要因がないため逆に安定 |
| 海水（pH8.3） | KH消費・酸性化との戦い | アルカリ域でロック可能（理論上） |

> 「どんなpHも維持しにくいのではなく、硝化サイクルがあるから維持できなかった。」

ホロビオント式の本質：**「炭素による従属栄養細菌のアンモニア同化 ＋ デトリタス食性生物による窒素固定化」**。この二段構えが成立する環境であれば、水質・水槽形式を問わず原理的に機能する。

### さらなる帰結：pH決定要因の完全外部化

ホロビオント式で内部pH変動要因（H+産生）がゼロになると、pHを決める要因は：

1. **底床素材の酸塩基性**
2. **足し水・立ち上げ水のpH**

…の2つだけになる。ここでpHニュートラルな底床素材（軽石・富士砂等）を選べば、残る変動要因は「入れる水のpH」のみ。

> **「立ち上げ時と足し水時に目標pHに調整した水を入れるだけで、以後一切の変動が起きない」**

アフリカンシクリッド（pH8.0キープ）なら立ち上げと足し水をpH8.0に合わせるだけ。KH補充も換水もpH調整剤も不要。pH調整剤・バッファー剤というアクア用品カテゴリが丸ごと不要になる。

## v8改訂に向けた用語の再定義リスト

ホロビオント式は「濾過」ですらない。既存の語彙を使うと概念がズレるため、v8改訂では冒頭に用語の再定義章を置くべき。

| 従来の言葉 | ホロビオント式における正確な言葉 | 理由 |
|-----------|--------------------------|------|
| 濾過 | **代謝** | 取り除くのではなく変換・循環させる |
| 濾過槽（サンプ） | **代謝槽** | 生態系の心臓部・生産の場 |
| 立ち上げ期間 | **バクテリア定着・遷移期間** | 硝化菌待ちではなくバイオフィルム形成の時間 |
| 浄化 | **物質変換** | 汚れを消すのではなく別の形に組み換える |
| フィルター | **代謝チャンバー** | 取り除く装置ではなく代謝環境 |

## ホロビオント理論の核心一文（v8冒頭候補）

> **「従属栄養細菌による炭素同化（アンモニア→バイオマス）＋ デトリタス食性生物による捕食（バイオマス→窒素固定化）＋ メインタンクへの還流（循環の完成）」**

この三段階が成立すれば実装は問わない。ベントス（イトミミズ・ヨコエビ）はその最小コスト・最適解だが、大量のドジョウでも、コペポーダでも、海綿でも理論は成立する。「誰がバイオマスを食べるか」は実装の選択肢であって、理論の要件ではない。
