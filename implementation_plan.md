# 日本語版 ホロビオントアクアリウム理論ページ — コンテンツ改修計画

カジュアル論文スタイルへのリライト。中立性の確保、論理構造の整理、重複の削除、視覚的改善を行う。

## 理論の核心（ユーザー確認済み）

本理論の基盤は以下の2層構造として整理する：

1. **理論（窒素循環の原理）**：従属栄養細菌によるアンモニアの炭素同化 → バイオマス蓄積 → デトリタス食性生物が捕食 → **デトリタス食性生物をメインタンクに戻し、肉食魚に捕食させる** → 窒素が食物連鎖内で循環。デトリタス食性生物の種類（ベントス・ドジョウ等）は理論上問わない。
2. **手段（実装の選択）**：デトリタス食性生物をメインタンクに戻す方法の違い。
   - **フルオート**：イトミミズ等の微小ベントスが通常サイズのポンプを通過 → メインタンクの魚が自動的に捕食（現実的な家庭用構成）
   - **手動 / 大型ポンプ**：ドジョウ等の大型デトリタス食性生物 → 人の手でメインタンクに戻すか、巨大なポンプを使用 → 大型肉食魚が捕食

3. **底床の位置づけ**：底床は自由選択であり、システムの前提条件ではない。底床素材がpHロックの平衡点を決めるパラメータとなる（pHロック理論と連動）。今回の弱酸性環境では焼赤玉土が最適だった。サンプに多孔質素材（バクテリアの定着面）さえあれば機能し、メインタンクはベアタンクでも理論上成立する。

> この「理論（原理）」と「手段（実装）」と「パラメータ（底床等）」の分離を文書全体で徹底する。

## 改修概要：問題点と対策

### 1. 中立性の欠如（トーン問題）

現状、個人ブログ調の主観的表現が多く、「カジュアル論文」としての中立性が不十分。

| 現状の表現 | 問題 | 改修方針 |
|---|---|---|
| 「面倒なのだ」「腰も痛くなる」 | 個人的感情 | 客観的な動機付け（大型水槽の維持管理コスト）に置換 |
| 「この壁をベントスの食物連鎖で突破した」 | 勝ち誇った表現 | 「ベントスの食物連鎖が代替手段となる」等に |
| 「驚異的な安定」「究極の状態」 | 最上級表現 | 「高い安定性を示した」等、事実ベースに |
| 「必要な設備のハードルが高く…胆力が求められる」 | 主観的 | 前提条件の客観的列挙に |
| 「俺もしない」 | 個人的 | 削除またはエピグラフとして明示的に分離 |

### 2. 理論と手法の混在

| セクション | 問題 | 改修方針 |
|---|---|---|
| 「三つの無」（Part 1内） | 実践的メリットが理論編に配置 | Part 1 の「システム原理」説明の一部として残すが、**理論的に導出される帰結**として再構成。実践上の注意事項はPart 2へ移動 |
| 「なぜ嫌気性ではなく好気性を選んだか」（Part 1内） | 設計思想（手法選定理由）であり純粋な理論ではない | セクションタイトルを「好気性アプローチの理論的優位性」に変更し、好気性条件下での観察可能性を理論的根拠として記述 |
| pHカウンター + 解説文（Part 2内） | Part 1で解説したpHロック理論の重複 | Part 2では実測値のみ提示し、理論的説明への相互参照に |

### 3. 重複した説明

| 重複箇所 | 対策 |
|---|---|
| 硝化バイパスの説明が(a)図の注記 (b)本文 (c)pH解説で3回 | 図で一度詳細に説明し、他はクロスリファレンスに |
| 「オーバーフロー必須」が序文・サンプ比較表・設備要件で3回 | 設備要件セクションで一度詳述し、他は簡潔に言及 |
| ベントスの役割が(a)系統図 (b)ループ図 (c)三つの無で重複 | それぞれの文脈で異なる側面を説明するよう整理 |

### 4. 論理的前提条件の不足

| 不足箇所 | 追加内容 |
|---|---|
| 「従属栄養細菌は硝化菌よりも圧倒的に速い」 | 増殖速度の理論的根拠（世代時間の差：従属栄養菌 ~20分 vs 硝化菌 ~12-24h）を追加 |
| 「アンモニア直接同化」の仕組み | C:N比に基づく同化条件（C:N > ~20 で同化優位）の説明を追加 |
| 「pHロック」 | 因果関係を明示: 従属栄養同化 → 硝化非進行 → HNO₃非生成 → pH低下なし → 底床の緩衝平衡点で安定 |
| 「ベントス」の位置づけ | 理論上はデトリタス食性生物全般で成立すること、ベントスが選ばれるのはポンプを通過できるサイズの手段選択であることを明記 |
| 「赤玉土」の位置づけ | 底床はシステムの前提条件ではなく、pHロックの平衡点を決めるパラメータであることを明記。サンプに多孔質素材があれば機能し、メインはベアタンクでも可 |
| 白濁→クリア化の過程 | バクテリアのフロック形成と定着の理論的説明を追加 |

### 5. 視覚的改善

| 改善箇所 | 内容 |
|---|---|
| 硝化バイパス図 | 「従来経路」と「本システム経路」の対比をより明確に。従来経路をグレーアウト/取消線で視覚的に「不活性」を表現 |
| C:N比の概念図 | 新規追加。炭素源投入→C:N比上昇→同化優位のフロー図 |
| Part境界 | 既存のPart境界は維持しつつ、Part内のセクション間遷移をよりスムーズに |
| 「前提条件」ボックス | 記事冒頭に「本理論の前提条件」として独立したカードを新設 |
